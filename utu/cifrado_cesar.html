¿Qué es la Criptografía?

La criptografía es la ciencia y arte de escribir mensajes en forma cifrada. Un mensaje codificado debe ser privado: solo quien envía y quien recibe tiene acceso al contenido.

Sus primeros usos eran militares. Aunque los sistemas antiguos se consideran obsoletos hoy, sus conceptos son la base de la criptografía moderna.

La Cifra de César — ROTN

Consiste en desplazar cada letra del abecedario N posiciones. La transformación se llama ROTN (ROTAR N posiciones) porque es un cambio cíclico: cuando llega a Z, vuelve a A.

Trabajamos con un alfabeto de 26 letras (sin Ñ, opcional). Solo mayúsculas. Los espacios se conservan.

Texto originalIMPULSO
ROT1JNQVMTP
ROT2KÑRWNUQ*
HOLA AMIGOS + ROT2JQNC COKIQU

*El ejemplo con Ñ es del enunciado; el cifrador de pruebas usa el abecedario de 26 letras inglés.

El disco de César — Rotación visual

Letra a cifrar

A (pos 0) + ROT3 = D (pos 3)
Resultado = (( c - 'A' + rot ) % 26) + 'A'
c − 'A' convierte la letra en un número (A=0, B=1 … Z=25). Se le suma rot. El operador % 26 asegura que si el resultado supera la Z, vuelva a empezar desde la A. Finalmente + 'A' convierte el número de vuelta en una letra.

¿Por qué el operador resto ( % ) nos salva?

Conceptos necesarios en Java

  • Transformar entero → letra (ASCII): (char)(65 + i) — A=65, B=66 … Z=90
  • String → array de caracteres: "HOLA".toCharArray()
  • Pasar a mayúsculas: texto.toUpperCase()
  • Largo del String: texto.length()
  • Concatenar chars: resultado += (char) nuevoValor
  • Fórmula de rotación cíclica: ((c - 'A' + rot) % 26) + 'A'
  • Desencriptar = rotar en sentido inverso: ((c - 'A' - rot + 26) % 26) + 'A'
  • Conservar espacios: verificar c == ' ' antes de transformar

Lógica del cifrado — paso a paso

// 1. Leer el texto y pasarlo a mayúsculas
String texto = sc.nextLine().toUpperCase();
 
// 2. Recorrer cada carácter
for (char c : texto.toCharArray()) {
 
// 3. Conservar espacios sin cambiar
if (c == ' ') { resultado += ' '; continue; }
 
// 4. Calcular posición original (0-25)
int pos = c - 'A';
 
// 5. Aplicar rotación cíclica
int nueva = (pos + rot) % 26;
 
// 6. Convertir de vuelta a char
resultado += (char)('A' + nueva);
}
